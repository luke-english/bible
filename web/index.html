Bible Learning System

This is a Bible Learning System which will help you study the Word of God in an engaging and interactive way. The box bellow is a demo of the WASM version of the Bible Learning System. Use your keyboard to type in the words you see on the screen.

How to use it

This software is free and open source. It comes in two different shapes a WebAssembly executable that runs on the browser, and a command line application.
  1. Everyone is free to use the WASM version on this website.
  2. Also, please feel free to install the WASM pacakage on any website you manage as component
  3. BibleTracker uses this software, and also it has many different enhancements, like tracking your progress
  4. You can run it from command line using pre-packed Docker image.
  5. This is an open source project, please build and customize it right from the source code.

Run it in a Docker Container

If you have docker installed, it is very easy to run it from the command line. You can use the following command: 
      docker run -it --rm text-based/bible

Install it on your web site

The WebAssembly version can be installed as a HTML component on your website. If you use some sort of bundler, the easiest way to install it to your project is via NPM: 
      npm install text-based/bible
then embed it in your HTML like: 
      <script>
        const container = document.getElementById('terminal-container');
        bible.loadWasm(container);
      </script>
